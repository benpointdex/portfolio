# Shakib | Software Engineer Portfolio

A minimal, high-fidelity personal portfolio built with React and Vite. Designed to showcase scalable backend systems, modern web applications, and thoughtful user experiences.

## Overview

This repository contains the frontend codebase for my personal portfolio. The interface was custom-built using a strict design system emphasizing modern typography (Inter & Playfair Display), subtle micro-interactions, and a clean, editorial layout.

### Featured Projects
- **AuthPulse**: A complete identity and access management solution built for modern web applications featuring stateless JWT security and Google OAuth2 integration.
- **Shortify**: A secure URL management tool designed to convert complex links into manageable assets.

## Tech Stack

- **Framework**: React + Vite
- **Styling**: Vanilla CSS with custom design tokens
- **Typography**: Inter (UI/Body) & Playfair Display (Display/Headers)

## Running Locally

To run the portfolio locally on your machine:

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Design Philosophy
The design language focuses on a muted, elegant color palette (Primary `#52563F`, Surface `#F4F2EB`) and highly controlled, cinematic micro-interactions.
